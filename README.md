## Dependencies

    - sdl2
    - sdl2-image
    - sdl2-mixer\[libvorbis]
    - sdl2-ttf

## Assets

- sound: [Sci-Fi Sounds](https://www.kenney.nl/assets/sci-fi-sounds)
- font: [Kenney Fonts - Kenney Future](https://www.kenney.nl/assets/kenney-fonts)

## Build and Run
